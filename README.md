# Smart-ShoppingCart
# 🛒 SmartCart Pro

A modern, responsive, and professional shopping cart web application built using **HTML, CSS, and JavaScript**.
This project demonstrates real-world frontend concepts including dynamic UI updates, state management, and user-friendly checkout flow.

---

## 🚀 Features

* ✅ Add items with name, price, and quantity
* 🗑 Remove items dynamically
* 💾 Persistent cart using LocalStorage
* 🌙 Dark / ☀️ Light theme toggle
* ⚠️ Custom alert notifications (styled, not default browser alerts)
* 🧾 Clean checkout flow with order summary
* 🔒 Cart locking after checkout (no edits allowed)
* 🔄 Start new order functionality
* 📱 Fully responsive design

## ⚙️ How It Works

### 1. Adding Items

Users can add products with:

* Name
* Price
* Quantity

### 2. Cart Rendering

Items are dynamically displayed in a table using JavaScript.

### 3. Local Storage

Cart data is saved in the browser so it persists after refresh.

### 4. Checkout Flow

* Clicking **Checkout** hides the cart
* Displays a clean **Order Summary**
* Calculates total with tax

### 5. Order Locking

Once checkout is triggered:

* Items cannot be modified
* Cart is locked
